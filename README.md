# Chat Cat  
  
This is a chat application written in Node.js, with socket.io and expressjs.  

## Install

To install and run locally ...

```sh
$ git clone https://github.com/skylerto/chat-cat.git
$ cd chat-cat/
$ npm install
$ node app.js
```
